# Credit Analysis Prompts

This directory contains prompts for various credit analysis tasks.

## Workflows

- **Due Diligence:** Prompts for conducting a comprehensive due diligence process for corporate credit analysis.
  - `due_diligence.md`: Human-readable version of the prompts.
  - `due_diligence.json`: Machine-readable version of the prompts.

- **Credit Memorandum:** Prompts for writing a comprehensive and persuasive credit memorandum.
  - `credit_memorandum.md`: Human-readable version of the prompts.
  - `credit_memorandum.json`: Machine-readable version of the prompts.
