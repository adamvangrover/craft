Sales & Business Development

Personalized Prospect Outreach

Drafts a highly personalized cold outreach email using CRM data and a prospect's LinkedIn profile to increase engagement.

You are an expert sales development representative with a friendly but professional tone.

Draft a personalized outreach email to {Prospect Name}, {Prospect Title} at {Company Name}. The goal is to schedule a 15-minute introductory call.

Use the following information:
1. Prospect's LinkedIn Profile summary: """{paste LinkedIn summary here}"""
2. Our CRM data for {Company Name}: "{paste CRM notes here}"

The email must be no more than 150 words, feature an engaging subject line, reference a specific point from their profile, and end with a clear call to action.

Competitor Battle Card Creation

Generates a concise "battle card" for a competitor, synthesizing internal data and external research into actionable talking points.

You are a strategic market intelligence analyst. Create a one-page competitor battle card for {Competitor Name}.

Base your analysis on:
1. Our internal CRM notes on deals lost to them in the last 6 months.
2. The attached market analysis report: "{report_filename.pdf}".
3. Publicly available information regarding their recent activities.

Provide the output in Markdown with sections for: Overview, Strengths, Weaknesses, Winning Strategy, and Recent News.

Write a concise executive summary explaining the top three largest variances between budgeted and actual spending by department. For each variance, state the department, the variance amount (in dollars and percentage), and a potential reason.

Python Code Generation

Generates a clean, efficient, and well-documented Python function based on a detailed set of requirements.

You are an expert Python developer following PEP 8 standards.

Write a Python function named `process_data` that takes a pandas DataFrame as input and performs the following steps:
1. Remove any rows with missing values in the '{column_name}' column.
2. Convert the '{timestamp_column}' column from a Unix epoch to a datetime object.
3. Create a new column called '{new_column_name}' based on another column.
4. Return the processed DataFrame.

The function must include a docstring, type hints, and a try-except block to handle potential KeyErrors.
