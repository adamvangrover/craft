# Venture Capital Module

This module provides a comprehensive guide to the Venture Capital industry, covering deal sourcing, term sheet negotiation, startup valuation, and exit strategies.

## Core Components

1.  **Deep Dives**: In-depth articles on key topics.
    *   Term Sheet Negotiation
    *   Startup Valuation Methods
2.  **Tools**: Interactive calculators and models.
    *   Cap Table Builder: Model ownership dilution across funding rounds.
3.  **Simulations**: (Planned) Negotiation scenarios.

## Learning Objectives

*   Understand the economics of VC funds.
*   Master the key clauses in a VC Term Sheet.
*   Apply valuation methodologies to pre-revenue startups.
*   Model capitalization tables and understand dilution impact.
