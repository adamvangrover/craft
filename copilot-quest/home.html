Overview

Copilot Quest is an interactive learning experience designed to test your knowledge and collaboration skills in a fun and engaging way. This project encompasses two main variations of the game, along with a rich library of resources for learning and development.

Games

  • Virtual Game - An automated, self-contained game experience.
  • In-Person Game - A version of the game designed to be facilitated in a live setting.

Materials

This repository contains a wide range of materials to support your learning journey. Explore the links below to access the full library of resources.

Usage in Banks and Risk

While Copilot Quest is a game, it has practical applications in the financial industry. It can be used to train analysts on risk management, compliance, and other critical topics in a simulated environment.
