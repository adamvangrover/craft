Introduction

This tutorial will show you how to use Copilot to write documentation more efficiently. We will cover the basics of generating documentation, as well as more advanced techniques like using comments to guide Copilot's suggestions.

Generating Documentation

To generate documentation with Copilot, simply start typing and Copilot will automatically suggest completions. You can accept a suggestion by pressing the Tab key.

For example, to create a simple README file, you could type the following:


# My Project

This is a simple project to demonstrate how to use Copilot to write documentation.

Copilot will likely suggest the entire README file after you type the first line.

Using Comments to Guide Copilot

You can use comments to give Copilot more context and guide its suggestions. For example, if you want to create a function with a docstring, you could write the following comment:


def add(a, b):
    """This function takes two numbers and returns their sum."""
    return a + b

Copilot will use this comment to generate the docstring for you.
